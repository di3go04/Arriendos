# Guion demo de 90 segundos

1. Abrir landing `/` y mostrar propuesta de valor.
2. Ir a `/demo` para explicar dashboard, contratos y pagos.
3. Entrar como arrendador demo.
4. Crear una propiedad.
5. Crear un contrato desde plantilla.
6. Mostrar firma electronica.
7. Mostrar pagos y reporte financiero.
8. Abrir portal publico `/propiedades`.
9. Enviar un lead de prueba.
10. Cerrar en `/admin/readiness` mostrando estado de producto.

Mensaje de venta: RentNow reduce tareas manuales de arrendamiento, centraliza pagos/documentos y esta preparado para white-label en inmobiliarias.
