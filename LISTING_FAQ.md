# Preguntas Frecuentes — Compra de RentNow

### 1. ¿Puedo ver una demo funcionando?

Sí. El vendedor tiene una demo desplegada. Solicita el enlace y podrás explorar la landing page, el dashboard, y las funcionalidades principales con datos de prueba.

### 2. ¿Cuánto cuesta operar esto mensualmente?

**$0/mes.** Todos los servicios tienen free tier suficiente para empezar: Supabase (500 MB), Vercel (100 GB), Mercado Pago (sin costo fijo), Stripe (sin costo fijo), Google Gemini AI (gratuito). No necesitas pagar nada para ponerlo en marcha.

### 3. ¿Necesito saber programar?

Necesitas nivel **básico a medio**. Saber usar la terminal, Git, y entender variables de entorno es suficiente. Todo el setup está automatizado con `npm run setup:demo`.

### 4. ¿Incluye soporte después de la compra?

Sí, el vendedor ofrece **1 mes de soporte gratuito** post-venta para ayudarte con la instalación, configuración y resolución de bugs. Pasado ese mes, puedes contratar soporte extendido.

### 5. ¿Puedo vender este SaaS a mis clientes (white-label)?

Sí. La licencia comercial te permite usar el código como base para tu propio producto SaaS y cobrar a tus usuarios. No puedes revender el código fuente tal cual como si fuera tuyo.

### 6. ¿Qué licencia incluye?

Es una **licencia comercial privada**. Obtienes el código fuente completo. Puedes modificarlo, desplegarlo y cobrar a tus usuarios. No puedes redistribuir el código original como producto propio.

### 7. ¿Por qué este precio?

El proyecto tiene **500+ horas de desarrollo**, stack moderno (Next.js 15, React 19), 3 pasarelas de pago, IA integrada, 44 módulos funcionales, y está listo para producción. En CodeCanyon productos similares se venden a $69–$199, y boilerplates comparables cuestan $199–$1,499. Como proyecto completo listo para SaaS, su valor es de **$8,000–$15,000**.

### 8. ¿Aceptan pagos en cuotas?

Depende del marketplace. En Acquire.com y Microns.io los pagos se procesan mediante el sistema de escrow de la plataforma. Consulta las opciones de pago disponibles al momento de la compra.
