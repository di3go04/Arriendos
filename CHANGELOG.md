# Changelog

## 0.1.0 (2025-01-15)

### Added
- Gestión completa de propiedades (CRUD con imágenes, mapa, estado)
- Generación de contratos con IA (Gemini) y plantillas personalizables
- Portal público de propiedades con leads
- Dashboard financiero con métricas en tiempo real
- Módulo de pagos: Mercado Pago, PayPal, Stripe
- Suscripciones SaaS con planes Free, Pro y Business
- Firma electrónica de contratos con registro de auditoría
- Multi-idioma: español, inglés, portugués, francés, alemán, italiano
- Multi-moneda: USD, COP, MXN, EUR, BRL
- Panel de administración con gestión de usuarios
- Integración con WhatsApp (Baileys) para notificaciones
- Notificaciones push y por email (Resend)
- Exportación a Excel y PDF
- Módulo de gastos con categorías
- Roles y permisos (RBAC)
- PWA con soporte offline
- Modo oscuro automático
- Onboarding interactivo multi-paso
- Página de estado del sistema
- Página 404 personalizada
- Documentación de API en /developers con Swagger UI
- Analytics con PostHog
- Docker Compose con 4 servicios
- SEO: sitemap dinámico, robots.txt, Open Graph, Schema.org

### Fixed
- Sanitización HTML con DOMPurify en vistas de contratos
- Manejo de errores en webhooks de pagos
- Enlaces internacionalizados en registro
